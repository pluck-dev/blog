- **콘텐츠 양산 자체 비용은 매우 저렴**: 1만 건당 약 ₩70만원 (LLM + 이미지)

## 4. 객체 사진 1회성 비용 (전략별)

|---|---|---|---|
| **A. 가맹 객체가 직접 업로드** | ₩0 | ★★ (등록 인센티브 기획) | ★★★★★ |
| **B. 자체 운영팀 촬영** | ₩2,000만~5,000만 | ★★★★ (시간 부담) | ★★★★ |
| **C. 외주 사진사 풀** | ₩5,000만~3억 | ★ (관리 쉬움) | ★★★★ |
| **D. AI 생성으로 전부 대체** | ₩30만 (Flux 1만장) | ★ | ★★ |

→ 가맹 없는 도메인이라면 **D + F 조합 (AI + 스톡)** 으로 시작, 월 10~30만원.

## 8. 비용 최소화 팁

콘텐츠 양산 비용을 더 줄이는 방법:

| 기법 | 절감 효과 | 트레이드오프 |
|---|---|---|
| **프롬프트 캐싱** (Anthropic) | LLM -40~70% | 캐시 갱신 주기 관리 필요 |
| **Haiku로 본문도 시도** | LLM -60% | 품질 검증 강화 필요 |
| **Stable Diffusion 셀프호스팅** | 이미지 -95% | GPU 셋업 부담 |
| **프롬프트 정교화로 재생성 ↓** | LLM -20~30% | 프롬프트 작성 시간 |
| **OpenAI Batch API** | LLM -50% | 24시간 지연 허용 시 |
| **자체 임베딩** (BGE-M3 셀프) | 임베딩 -100% | 추론 인프라 필요 |

→ **풀 최적화 시 1건 $0.05 → $0.015 (1/3)** 가능
→ 1만 건 ₩70만 → ₩20만 절감 가능

## 9. 비용 vs 시간 트레이드오프

발행 속도를 빠르게 하려면 비용이 늘어남:

| 일 발행 속도 | 동시성 | 월 발행 | 월 LLM 비용 | 권장 단계 |
|---|---|---|---|---|
| 일 30건 | 5 | 900건 | ₩6만 | MVP/검증 |
| 일 100건 | 10 | 3,000건 | ₩21만 | 정착 |
| 일 300건 | 30 | 9,000건 | ₩63만 | 본격 스케일 |
| 일 1,000건 | 80 | 30,000건 | ₩210만 + Rate Limit 우려 | 비추천 |

→ 운전선생의 2026년 1월 일 130건이 **최적점**으로 추정.
→ 너무 빠른 발행은 색인 거부 / 양산 신호 트리거.

## 10. 의사결정 체크리스트

투자 결정 전 다음 질문에 답해야 함:

- [ ] **객체 사진 전략**이 정해졌는가? (가맹 / 외주 / AI / 스톡 중 선택)
- [ ] **도메인의 키워드 검색량**이 충분한가? (전체 매트릭스 슬롯 ≥ 10,000)
- [ ] **인건비를 외주로 갈지, 정규로 갈지** 정해졌는가?
- [ ] **YMYL 영역**(의료·금융·법률)이 아닌가? (페널티 위험 차단)
- [ ] **사실 데이터 소스**가 확보되어 있는가? (자체 DB + 공공)
- [ ] **18개월 운영 자금**이 준비되어 있는가? (색인·노출까지 시간 필요)

위 6개 중 4개 이상 ✅ 가 안 되면 PoC 단계에 머무는 게 안전.
