## 2. 이번 주 진행 내용

### 2-1. 데스크탑 앱 (Electron + React, ~5,400라인)
비개발자가 클릭만으로 전 과정을 운영하도록 구성했습니다.

| 화면 | 기능 |
|---|---|
| **대시보드** | 고객사별 생산 현황 카드 |
| **잡 모니터링** | **실시간 진행률·로그·성공/실패 카운트, 작업 취소** (5초 자동 갱신) |
| **고객사 상세(7개 탭)** | 개요/기획/글유형·디자인/설정/축/슬롯/완성글 |
| **완성글 상세** | 본문 미리보기 + 비용·토큰 통계 |

### 2-2. 기획 → 생성 워크플로우 정비
- **다음 단계 자동 추천:** 기획 미입력 → 글유형 미선택 → 슬롯 없음 → 작성 → 완성글 확인 순으로 시스템이 안내.
- **디자인 템플릿 5종**(매거진형/비교형/지역추천형/체크리스트형/전환형) 선택 제공.

### 2-3. 웹 자료 기반 사실 검증 (핵심 신규)
- 생성 전 **검색엔진에서 해당 주제의 실제 자료를 수집**하고 출처 신뢰도(공식 vs 블로그)를 평가.
- 생성된 글에서 **자료에 없는 상호명을 자동 탐지**해 걸러냄 → AI가 지어낸 가짜 정보(할루시네이션) 차단.

### 2-4. 발행 포맷 export
- 완성글을 **HTML(디자인 적용) / 일반 마크다운 / Hugo / Next.js(MDX)** 4종으로 내보내기 → 어떤 사이트 환경에도 바로 투입 가능.
