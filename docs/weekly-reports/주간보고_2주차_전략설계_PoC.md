## 2. 이번 주 진행 내용

### 2-1. 시드 매트릭스 설계 (콘텐츠 주제 자동 도출)
"무슨 글을 쓸지"를 사람이 정하지 않고 시스템이 자동 도출하도록 설계했습니다.

| 축 (Axis) | 값 개수 | 예시 |
|---|---|---|
| 지역 | 20개 | 수원, 안산, 인천, 강남, 부산… |
| 키워드 | 15개 | 운전면허학원, 1종보통, 필기시험… |
| 인텐트 | 7개 | 비교 / 가이드 / 비용 / 후기 / 시험팁… |
| 페르소나 | 8개 | 직장인 / 대학생 / 사회초년생 / 주부… |
| 수정자 | 12개 | 최단기 / 비용절약 / 셔틀편리 / 야간반… |

→ 축을 조합(카르테시안 곱)하면 **164,345개 슬롯 자동 생성**. 각 슬롯에 검색량·경쟁도·템플릿 가중치로 **우선순위 점수 자동 부여** → "가성비 높은 주제부터" 생산 가능.

### 2-2. 프롬프트 라이브러리 구축
- 템플릿(지역 BEST5, 비교, 가이드, 비용 전략 등)별 **생성 프롬프트를 표준화**.
- 동일 템플릿이라도 슬롯 변수(지역·페르소나·수정자)에 따라 **본문이 전부 달라지도록** 설계 → 1주차에서 확인한 경쟁사의 "100% 유니크" 품질을 재현하기 위한 핵심 장치.

### 2-3. 단건 PoC 실증 (가장 중요)
실제 글 1건을 입력부터 발행본까지 **풀 파이프라인으로 생성**했습니다.

- **대상 슬롯:** "수원 + 직장인 + 셔틀편리 + 야간반" (월 검색량 2,400, 우선순위 78.5점)
- **흐름:** 슬롯 선택 → 입력 데이터(학원 5곳) → 프롬프트 렌더링 → 본문 생성 → 자동 검증 → 재생성 → 메타·Schema.org 생성 → 표·이미지·내부링크 치환 → 발행본 완성

| 단계 | 결과 |
|---|---|
| 1차 생성 | 검증 18/21 통과 (3개 항목 미달) |
| **자동 재생성(보강 프롬프트)** | **검증 21/21 전부 통과** |
| 총 실행 시간 | 약 8초/건 |
| 건당 비용 | **약 $0.04** (본문+메타+임베딩 포함) |

→ **품질 미달 시 시스템이 스스로 보강 재생성**하여 통과시키는 자가 교정 루프를 검증했습니다.

- **단가 검증:** 건당 약 $0.04 → 1만 건 생산 시 LLM 비용 약 **$400 수준**. 사업 단가가 충분히 낮음을 확인.
