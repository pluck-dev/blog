## 1. 이번 주 핵심 요약 (3줄)

- 국내 최대 양산형 SEO 사이트(운전선생, `drivingteacher.co.kr`)의 발행 글 **21,275건을 전수 수집·분석**하여 운영 메커니즘을 완전히 역설계했습니다.
- 핵심 결론: 이들의 양산은 **단순 복붙이 아니라 임베딩 기반 중복 차단 파이프라인**으로, 본문 중복률이 **0.005%**에 불과한 엔터프라이즈급 시스템입니다.
- 동일 구조를 **우리가 재현 가능하다는 점을 확인**했고, 목표 품질 수치와 데이터 기반 사업 타당성 근거를 확보했습니다.

---

## 2. 이번 주 진행 내용

### 2-1. 경쟁사 데이터 전수 수집
- 운전선생 블로그 **21,275건** 스크래핑 (제목·본문·메타·발행시각·네이버 동시발행 여부 포함).
- 발행 추이 분석: 2023~2026년, **2026년 1월 피크 시 일 평균 130건** 발행 확인.

### 2-2. 양산 품질 역설계 (가장 중요한 발견)
경쟁사가 "어떻게 2만 건을 찍어내면서도 검색 패널티를 피하는가"를 정량 검증했습니다.

| 검수 항목 | 중복률 | 해석 |
|---|---|---|
| 제목 | 0.24% | 거의 전부 유니크 |
| 메타 디스크립션 | 0.01% | 자동 생성 + 중복 차단 작동 |
| 본문 전체 (MinHash 유사도) | **0.005%** | 2만 건 중 사실상 중복 0건 |
| 같은 템플릿 내부 유니크율 | **100%** | 템플릿은 50개뿐인데 본문은 전부 다름 |

→ **결론:** 경쟁사는 ① 50개 템플릿 × ② 지역·키워드·인텐트 조합 시드 × ③ LLM 생성 + ④ 실시간 임베딩 중복 차단 + ⑤ 사후 디덕션의 5단 파이프라인을 운영 중. **이 수치를 우리 시스템의 목표 KPI로 채택**했습니다.

### 2-3. 노출/색인 전략 역설계
- 사이트맵에서 블로그를 **의도적으로 제외**(양산 신호 차단)하면서도, 색인 API 직접 호출로 색인을 가속하는 구조 확인.
- 모든 페이지에 검색엔진(구글·네이버) 완전 대응 메타 + 네이버 블로그 동시 발행으로 **백링크까지 자체 확보**하는 구조 파악.

### 2-4. 산출물 문서화
이번 주 분석 결과를 재현 가능한 형태의 내부 문서로 정리했습니다.
- 전략 플레이북 / 프롬프트 운용 전략 / 비용 분석 초안 / 산출물 인덱스.

## 3. 의미 및 기대 효과

- **사업 리스크 사전 검증:** "양산형 SEO가 실제로 검색 노출이 되는가"라는 핵심 가설을, 추측이 아니라 **실데이터 2만 건으로 검증**했습니다.
- **재현 난이도 확인:** 진입장벽이 "콘텐츠 양"이 아니라 **"중복 차단 품질"**에 있다는 점을 특정했고, 이는 우리가 기술적으로 충분히 따라잡을 수 있는 영역입니다.
- **목표 수치 확보:** 막연한 목표가 아니라 경쟁사 실측치(본문 중복률 0.005% 등)를 벤치마크로 설정.
